# er-diagram-endpoint
Generate ER Diagram automatically and expose it as PNG file on URL.

#### Test:
Run application and open http://localhost:10000/er

#### Uses:
- SpringBoot
- SchemaCrawler (https://www.schemacrawler.com)
- Graphviz (https://www.graphviz.org)
